A História da Web

A Web, como a conhecemos hoje, foi concebida por Tim Berners-Lee , um cientista da computação britânico, enquanto trabalhava no CERN (Organização Europeia para a Pesquisa Nuclear) em 1989. Berners-Lee propôs um sistema de hipertexto que permitiria aos pesquisadores compartilhar e acessar documentos de forma eficiente, independentemente de sua localização física. Ele desenvolveu o protocolo HTTP (Hypertext Transfer Protocol) e a linguagem de marcação HTML (Hypertext Markup Language), fundamentais para a criação da Web.

Em 1991, Berners-Lee lançou o primeiro site do mundo, que descrevia o próprio projeto da World Wide Web. Esse site pioneiro também incluía informações sobre como criar suas próprias páginas da Web e acessar documentos online. Em 1993, foi lançado o primeiro navegador web gráfico, o Mosaic, desenvolvido por Marc Andreessen e sua equipe na Universidade de Illinois. O Mosaic foi uma ferramenta revolucionária que tornou a Web mais acessível e fácil de usar para o público em geral. A partir daí, a Web cresceu rapidamente, com a criação de milhares de sites em todo o mundo. Empresas e organizações começaram a reconhecer o potencial da Web como uma plataforma para comunicação, comércio e colaboração. Surgiram os primeiros motores de busca, como o Yahoo! e o Altavista, para ajudar as pessoas a encontrar informações na crescente massa de conteúdo online.

Atualmente, a Web continua a evoluir com o crescimento do comércio eletrônico, streaming de vídeo, inteligência artificial e Internet das Coisas (IoT). A proliferação de dispositivos conectados à Internet está criando novas oportunidades e desafios, desde questões de segurança cibernética até preocupações com privacidade e acesso equitativo.

Desde suas origens humildes até se tornar uma parte essencial da vida cotidiana para bilhões de pessoas em todo o mundo, a Web continua a moldar nosso mundo de maneiras profundas e inesperadas. À medida que avançamos para o futuro, é importante reconhecer o poder e a responsabilidade que acompanham essa tecnologia e trabalhar juntos para moldar um ambiente online que seja inclusivo, seguro e acessível para todos.

Introdução ao HTML

Inicio

HTML, que significa HyperText Markup Language (Linguagem de Marcação de Hipertexto), é a espinha dorsal da World Wide Web. É a linguagem de marcação fundamental usada para criar e estruturar páginas da web. Desde sua criação por Tim Berners-Lee no início da década de 1990, o HTML evoluiu para se tornar uma ferramenta essencial para desenvolvedores web e designers, permitindo a criação de páginas e aplicativos web dinâmicos e interativos. Consiste em uma série de elementos ou tags que são usados para definir diferentes partes da página, como cabeçalhos, parágrafos, imagens, links e muito mais. Cada elemento HTML é representado por uma tag delimitada por "<" e "/>".

Tags HTML

Inicio

Em HTML, as tags são elementos fundamentais usados para estruturar e definir o conteúdo de uma página da web. Elas são cercadas por colchetes angulares < > e geralmente ocorrem em pares, com uma tag de abertura e uma tag de fechamento. As tags de abertura são usadas para iniciar um elemento, enquanto as tags de fechamento são usadas para encerrar o elemento.

Aqui está uma breve explicação sobre os diferentes tipos de tags em HTML:

  1. Tags de Estruturação de Página:

    • html: Define o início e o fim de um documento HTML.
    • head: Contém metadados sobre o documento, como o título da página e links para folhas de estilo CSS e scripts JavaScript.
    • title: Define o título da página que é exibido na barra de título do navegador.
    • body: Contém todo o conteúdo visível da página, como texto, imagens, links, formulários, etc.

  2. Tags de Texto:

    • h1, h2, h3, h4, h5, h6: Usadas para definir títulos de diferentes níveis de importância.
    • p: Define um parágrafo de texto.
    • strong: Define texto importante ou enfatizado.
    • hr: Essa tag não necessita de fechamento, ela forma uma linha horizontal.
    • i: Transforma o conteudo em itálico.

  3. Tags de Lista:

    • ol: Define uma Lista ordenada.
    • ul: Define uma lista não ordenada.
    • li: Define um item de lista dentro de "ol" ou "ul".

  4. Tags de Lnks e Imagem:

    • a: Define um hiperlink para outra página ou recurso.
    • img: Define uma imagem a ser exibida na página.
